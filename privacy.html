Privacy Policy for GitHub Developer Assistant

Last updated: June 6, 2026

GitHub Developer Assistant is a browser extension designed to help developers and recruiters visualize GitHub profiles, track repositories, analyze statistics, and generate AI-powered insights using the Google Gemini API.

1. Data Collection and Usage

We believe in absolute data privacy. Here is how your data is handled:

  • GitHub Access Tokens: If you optionally supply a GitHub Personal Access Token to raise rate limits, it is stored locally on your device using Chrome's secure storage API (chrome.storage.local). It is only sent directly to official GitHub API endpoints (api.github.com) to authenticate requests.
  • Gemini API Keys: If you supply a Google Gemini API Key, it is stored locally on your device. It is only sent directly to Google's official Generative Language API endpoint to retrieve AI insights.
  • Favorites List: Any profiles you save as favorites are saved locally on your machine.
  • No External Trackers: The extension does not collect, track, or share any personal data, usage telemetry, or browsing history with third parties or external servers.

2. Permissions Requested

  • storage: To persist your light/dark theme preference, saved favorites, GitHub token, and Gemini API key across browser sessions.
  • https://api.github.com/*: To query developer profile details and repository data.
  • https://generativelanguage.googleapis.com/*: To call the Gemini API for profile summaries, repo explanations, and suggestions.

3. Third-Party Services

By using the optional AI features, data (specifically the developer's public profile details, bio, and list of public repositories) will be processed by Google's Gemini API according to Google AI Studio's terms of service.

4. Contact

For any questions or suggestions regarding this privacy policy, please contact us via the GitHub Developer Assistant repository page.
